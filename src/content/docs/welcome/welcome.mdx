Kilt is a mod that allows users to use Neo/Forge mods on Fabric Loader.

This is a bridge mod that allows you to add Neo/Forge mods into a Fabric instance, and have it work just like if it was a normal Fabric mod.

:::note{icon="heart"}
**Kilt is not a replacement for standard Fabric mod ports!**
:::
