Kilt is a mod that allows users to use Neo/Forge mods on Fabric Loader.

Kilt is classified under a bridge mod, which allows you to play mods from Neo/Forge on Fabric loader, which attempts to seem like it is working on Fabric natively.

:::note{icon="heart"}
**Kilt is not a replacement for standard Fabric mod ports!**
:::

{/* Icons reference: https://starlight.astro.build/reference/icons/#all-icons */}

This wiki goes over the [Installation](/guides/installation) of Kilt, available [JVM Arguments](/technical/jvm-args), and an [FAQ](/faq/how-work), along with a [Mod list](../mods).

Sometimes issues may arise. If you seem to be running into issues, refer to [Help](/welcome/help) for tips on how to get help. You can also look at [AnswerOverflow](https://www.answeroverflow.com/c/1062715218087645215) and find if anyone else has had the same issues as you.
