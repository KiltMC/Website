Installation Guide

To install Kilt, you’ll have to get some mods that Kilt requires on top of Kilt.

  1. You can install Kilt Nightly 1.21.1 or 1.20.1 or a release version of Kilt here. When picking the version, Kilt’s versioning is v(MC).x, So it would be v20.1.0. for the first version of 1.20.1.
  2. You can then put Kilt into your mods folder, and carry onto the rest of the guide. The mods required to install Kilt are:
  • Fabric API
  • Fabric Language Kotlin

We also recommend getting Mod Menu to allow editing of some mod configs in-game, on top of seeing if the mod loaded without checking logs.

  1. Now that you have all required mods, along with Kilt installed, install any version compatible Forge / NeoForge mod directly into the mods folder.
  2. Launch the game.
  3. All done! Kilt will now start making Neo/Forge mods run on Fabric automatically.
